## üìÇ Project Structure

The main application source code is located in `market/aura-web`.

-   `src/components`: Reusable UI components (Hero, Navbar, Cart, AuthModal).
-   `src/pages`: Main application views (Home, Shop).
-   `src/context`: Global state management.
-   `src/assets`: Static assets and images.

## üöÄ Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/tilak76/aura-market.git
    cd aura-market/market/aura-web
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
